Frequently Asked Questions

Here are some of the frequently asked questions.

What is roadmap.sh ?
roadmap.sh is a community effort to create learning paths, guides, project ideas and other similar content to help developers grow in their careers.
What are the plans for roadmap.sh ?

The plans for roadmap.sh involve expanding the roadmaps to include more roles and industries.

This includes introducing roadmaps for emerging technologies and domains such as AI, blockchain, and cloud computing.

The platform also aims to increase community participation by allowing users to contribute and suggest improvements.

How is roadmap.sh built ?

Roadmap.sh is built using a combination of React for the frontend, Node.js for the backend, and is hosted on Vercel.

The project uses GitHub for version control and community contributions, with contributions welcomed from anyone through pull requests.

Can I use roadmap.sh in my team ?

Yes, roadmap.sh can be used in teams.

It's a great resource for creating learning paths for team members, aligning their skills development with the company's goals, and onboarding new hires by providing them with clear learning roadmaps.

How can I create custom roadmaps ?

You can create custom roadmaps by forking the roadmap.sh GitHub repository and modifying the existing files to tailor them to your specific needs.

You can add your own roles, technologies, and learning paths that align with your team or personal goals.

Is roadmap.sh really 7th most starred project on GitHub ?

Yes, roadmap.sh is one of the most popular repositories on GitHub, ranking as the 7th most starred project as of today.

The project has gained immense popularity due to its community-driven approach, ease of use, and the value it provides to developers globally.
